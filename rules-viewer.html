Firestore Rules Viewer

Read-only view of every Firestore collection's access policy. Source: Central Hub/firestore.rules

This page is read-only. Firestore rules cannot be edited from the browser — they live in Central Hub/firestore.rules and deploy via firebase deploy --only firestore:rules. To request a rule change: edit the file, push to GitHub, deploy. To change a user's access without touching rules: use User Console (role assignments) or Page Access Manager (per-page sub-role gating).
